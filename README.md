# clean-todo

## 목적

사이드 프로젝트에 여러 기술을 도입해 봄으로써 새로운 지식을 습득

## 사용 기술

- React
- Redux
- NodeJs
- REST API
- ORM
- JWT
- Mocha(for test)
- present-container pattern

## 기능

1. 일반 로그인
2. 사용자가 TODO를 등록,수정,삭제
3. 등록한 TODO를 타임라인으로 정리

## 개발환경 세팅

### 모듈 설치

```bash
# API 서버
cd node-server
yarn

# 클라이언트
cd webapp
yarn
```

### 실행


```bash
# API 서버
cd node-server
yarn start

# 클라이언트, 실행후에 i를 입력하면 아이폰 가상 머신 실행
cd webapp
yarn start

```
